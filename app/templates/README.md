VM Boilerplate
==============

To use locally, you need:

- [ruby](https://www.ruby-lang.org/en/downloads/)
- [bundler](http://bundler.io/)
- [vagrant](http://www.vagrantup.com/) (optional)

To install:

	$ bundle install
	$ bundle exec librarian-chef install

Vagrant setup:

	$ vagrant plugin install vagrant-omnibus
